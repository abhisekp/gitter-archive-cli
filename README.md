# Gitter Archive

Gitter Chat room message downloader

# Usage

```sh
# run build
$ yarn run build

# start gitter archive
$ yarn run start
```

# Development

```sh
$ yarn global nodemon
$ nodemon -q -x yarn run start:dev
```

**OR for build watch**

```sh
$ yarn global nodemon
$ nodemon -q -x yarn run build 
```
